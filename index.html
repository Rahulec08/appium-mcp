‚ö° Quick Start Guide

1 Install the package
npm install mcp-appium-visual
2 Configure MCP Server (Claude Desktop)
{ "mcpServers": { "mcp-appium": { "command": "node", "args": ["./node_modules/mcp-appium-visual/dist/claude-desktop-server.js"], "env": { "APPIUM_SERVER": "http://localhost:4723" } } } }
3 Initialize Appium Session
import { initializeAppium } from 'mcp-appium-visual'; await initializeAppium({ platformName: 'Android', deviceName: 'emulator-5554', appPackage: 'com.example.app', appActivity: '.MainActivity', automationName: 'UiAutomator2' });
4 Start Automating with AI
// AI-powered element interaction await smartTap('Login Button'); // Visual element recovery await visualElementRecovery({ screenshotPath: './screenshot.png', elementType: 'button', expectedText: 'Submit' }); // Smart input with visual validation await smartAction({ action: 'sendKeys', selector: 'username_field', text: 'john.doe@example.com' });
